# Budget Tracker App

## Description
The Budget Tracker is a web application that helps users manage their personal finances by tracking income and expenses. It calculates totals, displays balance, and stores data persistently using localStorage.

## Features
- Add income and expense transactions
- Delete transactions
- Persistent data storage using localStorage
- Color-coded transactions (Green = Income, Red = Expense)
- Filter transactions (All / Income / Expense)
- Displays total income, total expenses, and balance
- Visual representation using a pie chart

## Technologies Used
- HTML
- CSS
- JavaScript
- Chart.js
